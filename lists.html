Списки

List – HTML-структура, которая позволяет группировать связанный контент
<ul> -- unordered list – неупорядоченный список
Каждый элемент ссписка должен быть окружен <li> тегом (list item – пункт списка)
Пример Git: HTML_tgsTraing1 – lists.html
Внутри <ul> не позволено писать текст, только пункты списка
<ol> -- упорядоченный список
Очень часто списки используются для структуризации навигации на веб-странице
