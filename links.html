Гиперссылки и ссылки

Internal links (внутренние ссылки)

Ссылки создаются с помощью спецификации тега <a> атрибутом href (hypertext reference)
Пример <a href="lists.html" title="Списки"> Тут я практиковал списки. :) </a>
Значение href’а может быть либо relative (относительным), либо абсолютным URL. В этом примере использован относительный URL (тк мы ссылаемся на страницу на нашем же сайте), так как это внутренняя ссылка. Конечно, мы можем указать и абсолютный URL, если он ведет на страницу на этом же сайте.
Title – атрибут, нужный чтобы при наведении курсора появилось краткое описание того, что лежит за гиперссылкой.
Контент внутри тега <a></a> представляет собой контент, по нажатию на который мы переходим по гиперссылке.

External links (внешние ссылки)

По сути все то же самое, но в href’e находится ссылка на внешний ресурс.
Однако есть фича, которая используется для внешних ссылок, и это атрибут target. Можно сделать например target = "_blank" – что заставит браузер открыть ссылку в новой вкладке.
Пример:
<a href="https://github.com/Vaclav-Kasimov/" target="_blank" title="github">

Fragment Identifier (идентификатор фрагмента)

Это гиперссылка на секцию нашей страницы. Может в виде ссылки принимать #+название атрибута id или например name какого-то участка нашей страницы. Пример в https://github.com/Vaclav-Kasimov/HTML_tgsTraing1/blob/main/links.html
Сейчас Fragment Identifier крайне востребован, так как используется для навигации внутри SPA(Single page application – приложение одной страницы)
