Модели контента

Content-model – этот термин описывает то, как браузер будет обращаться с элементами, принадлежащими к конкретной модели контента (content-model) и какие правила вкладывания к этим элементам будут применяться. То етсь какие элементы могут быть вложены (nested) в другие элементы. По спецификации HTML элементы могут быть либо блоковыми либо линейными. HTML5 делит эти две модели контента на 7 моделей.
Базовые модели:
Блоковые элементы
  • Начинаются с новой строки (если css-ом это не изменено)
  • Могут содержать внутри себя другие линейные или блоковые элементы
  • ГРУБО ГОВОРЯ это Flow Content (потоковый контент) по категоризации HTML5
Линейные элементы
  • Отображаются в одной строке по умолчанию
  • Внутри себя могут содержать ТОЛЬКО другие линейные элементы
  • ГРУБО ГОВОРЯ это Phrasing content (фразовый контент) по категоризации HTML5
Разделение на эти два типа остается актуальным (хотя есть 7 типов пятой версии HTML) так как такое разделение хорошо вяжется с правилами CSS.
Div – division – разделение
Span – интервал, диапазон
Это наиболее общие элементы для этих двух категорий. Div – блочный, span – линейный
