Настройка окружения. Подготовка к работе.

Git

Работы с гитом ведутся в командной строке (cmd из нужной директории).
Git clone + URL (пример git clone https://github.com/Vaclav-Kasimov/coursera_test1.git). Копирует в текущую директорию репозиторий по ссылке.
Git add + директория или файл (елси просто точку поставить вместо имени директории то пометит все) Это мы помечаем чтото чтобы закоммитить что-то в наш гит локально чтоб работать с ним
Чтобы уже закоммитить по-нормальному git commit -m(пометка для сообщения) “Сообщение”
И потом git push чтобы загрузить изменения на гитхаб
Git pull – апдейт локальных файлов в соответствии с удаленными

Browser-sync

Browser-sync start(стартует) --server(в режиме сервера HTTP на локальной машине) --directory(покажет лист директорй папки где я щас нахожусь) --files(какие файлы искать браузер-синку (для обновления например)) "**/*"(звездочка значит что обновлять ему надо все файлы слеш звезда то есть везде)
